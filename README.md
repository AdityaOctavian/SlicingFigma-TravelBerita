Liburan Indonesia(UNTUK TUGAS LKS)

Website sederhana bertema pariwisata Indonesia yang menampilkan informasi destinasi populer seperti Bali, Yogyakarta, dan Banda Neira. Project ini dibuat menggunakan HTML dan CSS murni dengan tampilan modern dan responsif.

 Fitur Utama
 Navbar transparan (glassmorphism) dengan efek blur modern
 Hero section dengan background gambar full screen
 Card destinasi wisata berisi gambar dan deskripsi singkat
 Desain aesthetic dengan overlay gelap agar teks tetap terbaca
 Responsive design (menyesuaikan layar HP & desktop)
 Hover effects interaktif pada menu dan card
